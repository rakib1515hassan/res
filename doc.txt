
1. Download Python version(3.10.6+)
2. Install pip
3. Install virtaulenv using commnad 
    pip install virtaulenv
4. Active Virtaulenv using `source venv/bin/activate` this commnad
5. Run 'pip install -r requirements.txt`
6. Run 'python3 manage.py makemigrations`
6. Run 'python3 manage.py migrate`
7. Create a superuser using 'python3 manage.py createsuperuser`
